# zxcvbn-ts-demo

A little demo of the [`zxcvbn-ts`](https://zxcvbn-ts.github.io/zxcvbn/guide/getting-started/) library.

## Running the demo

```bash
# install pnpm if you don't have it
npm install -g pnpm

# run the demo
pnpm install
pnpm dev
```

Local: http://localhost:5173/
